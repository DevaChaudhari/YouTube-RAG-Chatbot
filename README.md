# 🎥 YouTube RAG Chatbot

An AI-powered chatbot that answers questions from YouTube video transcripts using Retrieval-Augmented Generation (RAG).

## 🚀 Features
- Extracts transcripts from YouTube videos
- Splits text into meaningful chunks
- Generates embeddings using Google Gemini
- Stores embeddings in FAISS vector database
- Retrieves relevant context for user queries
- Answers questions using LLM

## 🛠 Tech Stack
- Python
- LangChain
- Google Gemini API
- FAISS
- YouTube Transcript API
